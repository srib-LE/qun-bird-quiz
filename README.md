# 群友是啥鸟？🪶

一个不太正经但莫名很准的群聊人格测试——15 道题 × 5 大维度 × 20 种鸟类结果。

> 看看你在群聊里的灵魂，住着哪只鸟？

## 玩法

回答 15 道关于群聊行为的灵魂拷问，系统从**活跃度、影响力、表达欲、归属感、战斗力**五个维度分析，匹配 20 种鸟类人格。

选择后自动跳转，不到 2 分钟出结果。

## 技术

纯静态 HTML 单文件，打开即用。支持键盘操作（数字键 1-4 选题、方向键翻题），移动端适配。
